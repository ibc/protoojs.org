protoo

Fast and extensible Node.js signaling framework for Real-Time Communication applications

$ npm install protoo --save

The API you know

Love Express? Protoo provides the very same API for processing and routing messages from your clients.

Use existing middlewares or build your own ones.

Not just for the Web

Connect browsers and mobile apps with Protoo. Use the same signaling protocol for all your clients.

Real-time communication for all.

Chat, WebRTC and more

Design your own protocol features. Basic ones such as messaging and audio/video calls are provided out of the box.

protoo-client library for browsers will be available soon.
